## 常量

常量应使用大写加下划线：
```python
MAX_THREADS
```
## 私有变量

私有变量应使用一个前缀下划线：
```python
_observers
```

##函数

函数应使用小写加下划线：
```python
get_names
```

## 模块名称

模块名称应使用全小写的名称，如果需要也可以使用下划线：
```python
requests
```

## 错误名称

错误名称应包含Error：
```python
ProtectedError
```

## 类名称

类名称应使用标题式命名：
```python
Base
```

## 类型变量

类型变量的名称应使用标题式命名，也可以加上_co或_contra等后缀：
```python
Nim
```
